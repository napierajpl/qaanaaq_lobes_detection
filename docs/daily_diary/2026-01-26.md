- Added `_match_size()` method to interpolate decoder features to match encoder skip connection sizes
- Added final upsampling step to ensure output matches input spatial dimensions
- Used bilinear interpolation for smooth resizing

## Issues Encountered & Resolved

1. **Spatial Dimension Mismatch** ✅
   - **Error**: `RuntimeError: Sizes of tensors must match except in dimension 1. Expected size 16 but got size 8`
   - **Cause**: SatlasPretrain encoder outputs features at different spatial scales than expected
   - **Fix**: Added interpolation-based size matching in decoder skip connections

2. **Output Size Mismatch** ✅
   - **Error**: `RuntimeError: The size of tensor a (64) must match the size of tensor b (256)`
   - **Cause**: Final decoder output didn't match input spatial dimensions
   - **Fix**: Added final upsampling step to match input size
