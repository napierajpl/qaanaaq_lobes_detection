# README assets

Images in this folder are referenced by the main project README.

**Optional assets** (copy from your diary plots or save from a training run if you want them to appear in the README):

- `representative_tile_layers.png` – Representative tile showing input channels and a good prediction (e.g. car tracks on Slope Stripes). Example source: diary `plots/2026-02-24_tile_7370.png` or any run’s best-tile visualization.
- `training_loss_example.png` – Simple loss curve during training (in progress). Example source: diary `plots/2026-03-05_loss_a307c8eb.png` or any run’s loss plot.
- `training_loss_detailed_example.png` – Detailed loss plot with early-stop note. Example source: diary `plots/2026-03-05_loss_e66b5253.png` or equivalent.

If these files are missing, the README will show broken image links until you add them.
