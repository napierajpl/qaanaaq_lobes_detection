# Agent instructions

Project conventions and rules live in **`.cursor/rules/`**:

- **core.mdc** — Always applied: principles, workflow, tooling, CI, git safety, communication.
- **python-and-scripts.mdc** — When working with `**/*.py`: code style, file size, script refactoring.
- **project-tracking.mdc** — When relevant: daily diary and improvements backlog locations.

Code quality beats speed. If rules conflict or something is unclear, stop and ask.
